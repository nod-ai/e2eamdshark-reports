# How to Send GPU vs CPU Comparison Reports via Email

The `gpu_vs_cpu_email.html` file in this folder is specifically designed to be email-friendly and will render correctly in all major email clients.

## Method 1: Copy HTML Content Directly (Recommended)

1. Open `gpu_vs_cpu_email.html` in a text editor
2. Copy the entire HTML content
3. In your email client, switch to "HTML mode" or "Source view"
4. Paste the HTML content
5. Send the email

### For Gmail:
- Compose a new email
- Click the three dots (⋮) → "Default styling"
- Use a browser extension like "Insert HTML" or copy-paste from browser preview

### For Outlook:
- Compose a new email
- Click "Insert" → "Attach File" → "Browse This PC"
- Select `gpu_vs_cpu_email.html`
- Right-click the attachment → "Insert as Text"

## Method 2: Open in Browser and Copy

1. Open `gpu_vs_cpu_email.html` in your web browser
2. Select all content (Ctrl+A / Cmd+A)
3. Copy (Ctrl+C / Cmd+C)
4. Paste into your email body (the formatting will be preserved)

## Method 3: Send as Attachment

Simply attach `gpu_vs_cpu_email.html` to your email. Recipients can open it in their browser to view the formatted report.

## Features in the Email Version

- **Side-by-side tables**: Passing and Fail summaries displayed next to each other
- **Color coding**: Green headers for passing tests, red for failing tests
- **Alternating row colors**: Improves readability
- **Metadata**: Shows when each report was generated
- **No external dependencies**: All styles are inline (no external CSS)
- **Mobile responsive**: Adapts to different screen sizes

## Automated Updates

- Always contains the latest comparison data
- Includes all three report types (ONNX, HuggingFace Top1K, ONNX Dup)

## Tips

- For best results, send the email as HTML (not plain text)
- Test by sending to yourself first to verify formatting
- If colors don't show up, check your email client's settings for displaying HTML emails
- The file is ~18KB, well within email size limits
